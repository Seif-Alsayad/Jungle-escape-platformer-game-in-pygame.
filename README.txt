PLATFORMER GAME MADE BY SEIF ALDIN ALSAYAD MATIC. NO. 100006546.

GAME CONCEPT:

You are a player in a platformer jungle game.
There are gems to collect and a key you need to collect to escape.
You can move left or right by using the arrow keys and jump by using the space key.
There is a trap that may come on your way occasionally.
The level is randomly generated every time you start the game.
If you win or die you can choose if you want to replay the same level again or randomly generate a new level grid.

HOW TO RUN THE GAME:
1. Download the zip file then extract it before running the game.py.
2. Make sure the game.py file is in the same directory as the assets folder so the game can pull the assets correctly.
3. Run the game using python or visual studio code.
